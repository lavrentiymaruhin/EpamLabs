2.(Проверка актуальности страницы)
2.1 Зайти на сайт http://www.yvr.ca/en/passengers
2.2 Навести на выпадающий список "FLIGHTS"
2.3 Нажать на кнопку "Departing Flights"

OP: Убеждаемся что открылась страница с поиском отправляющихся рейсов

3.(Поиск прибывших рейсов по городу)
3.1 Зайти на сайт http://www.yvr.ca/en/passengers
3.2 Навести на выпадающий список "FLIGHTS"
3.3 Нажать на кнопку "Arriving Flight"
3.4 Ввести в поле поиска "Los Angeles"

OP: Убеждаемся что поиск был выполнен по городам

4.(Поиск отправившихся рейсов по номеру рейса)
4.1 Зайти на сайт http://www.yvr.ca/en/passengers
4.2 Навести на выпадающий список "FLIGHTS"
4.3 Нажать на кнопку "Departing Flights"
4.4 Ввести в поле поиска "DL4685"

OP: Убеждаемся что поиск был выполнен по номеру рейса

5.(Поиск рейсов с местом назначения (ошибка))
5.1 Зайти на сайт http://www.yvr.ca/en/passengers
5.2 Навести на выпадающий список "FLIGHTS"
5.3 Нажать на кнопку "Flight Schedules"
5.4 Убедится что открылась нужная страница
5.5 Ввести в поле "To" значение " "
5.6 Нажать кнопку "Search"

OP: Выдаст сообщение "Please choose a destination"

6.(Поиск по сайту)
6.1 Зайти на сайт http://www.yvr.ca/en/passengers
6.2 Вводим в поле поиска "Business"
6.3 Нажать на кнопку поиска

OP: Выдаст результаты поиска по сайту со словом "Business"

7.(Поиск магазинов)
7.1 Зайти на сайт http://www.yvr.ca/en/passengers
7.2 Нажать на кнопку "Shop, Dine and Services"
7.3 Перейти на вкладку "Shop"
7.4 Выбрать терминал "Canada"
7.5 Ввести в поле поиска "coastal"
7.6 Выбрать Security "Before"
7.7 Нажать кнопку "Search"

OP: Найдет магазины с терминалом канада и ключивым словом "coastal"

8.(Поиск отеля)
8.1 Зайти на сайт http://www.yvr.ca/en/passengers
8.2 Нажать на кнопку "Shop, Dine and Services"
8.3 Перейти на вкладку "Services"
8.4 Выбрать Services type "Hotels & Conference Facilities"
8.5 Нажать кнопку "Search"

OP: Список всех отелей

9.(Смена языка)
9.1 Зайти на сайт http://www.yvr.ca/en/passengers
9.2 Нажать на текст "Français"

OP: Откроется сайт на французком языке

10.(Поиск обменных пунктов)
10.1 Зайти на сайт http://www.yvr.ca/en/passengers
10.2 Нажать на кнопку "Shop, Dine and Services"
10.3 Перейти на вкладку "Services"
10.4 Выбрать Services type "Banking & Currency"
10.5 Поставить отметку на "Open now"
